Funções necessárias para que o jogo funcione:
1 - Realize o download do repositório;
2 - Carregue o repositório através do Visual Studio;
3 - Inicie o código clicando em "Iniciar".

Conjunto de atividades para criar o jogo
1 - Receber o jogador;
2 - Checar se o jogador deseja ler as regras;
3 - Se S, informar o jogador das regras. Senão, iniciar o jogo;
4 - Iniciar o gerador de números;
5 - Sortar um número de 1 a 20;
6 - Definir o TARGET como o número sorteado;
7 - Somar o valor do TARGET à variável TOTAL;
8 - Definir as variáveis necessárias para o palpite do jogador (bool isNumber e int guess);
9 - Pedir o palpite do jogador;
10 - Checar se o palpite do jogador é válido (se for um int, é válido. Senão, pedir que ele insira um novo valor)
11 - Criar um loop em que o valor TOTAL é checado (se é igual ou maior que 100);
12 - Checar se o palpite do jogador GUESS é igual ao TOTAL;
13 - Se sim, anunciar a vitória do jogador;
14 - Se não, pedir um novo palpite;
15 - Se TOTAL for maior ou igual a 100, declarar a derrota do jogador;
16 - Pedir que o jogador clique na tecla ENTER para encerrar o jogo;
17 - Encerrar o jogo após o input.
